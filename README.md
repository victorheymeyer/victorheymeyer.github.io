Source for my personal website, [heymeyer.com](https://heymeyer.com), a home for data and analytics projects.

Main Project:  
**Seattle Jobs** – daily snapshots of company job boards, filtered for Seattle and remote roles. 

Each project follows the same low-cost pattern: a scheduled GitHub Actions job pulls data from a public source, stores daily snapshots in Supabase, and a static page reads from Supabase to show summaries, trends, and interactive views. Hosted on GitHub Pages.
